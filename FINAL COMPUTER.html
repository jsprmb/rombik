Computer Case
A computer case, also known as a chassis, is the enclosure that houses and protects the computer components. It provides structural support and helps in cooling the components.

List of computer case sizes (known as form factor):
  • Very small form factor: Supports Mini ITX motherboards
  • Small form factor: Supports micro ATX motherboards.
  • Standard form factor: Supports standard ATX motherboards.
  • Larger form factors: Supports ATX and XL-ATX motherboards.

CPU (Central Processing Unit)
The CPU is the brain of the computer, responsible for executing instructions and performing calculations.

RAM (Random Access Memory)
RAM is a type of memory that stores data that the CPU needs to access quickly. It provides temporary storage for running applications and data.

GPU (Graphics Processing Unit)
A GPU is a specialized electronic circuit that accelerates the creation and rendering of images, video, and animations.

Sound Card
A sound card is an expansion card that provides audio input and output capabilities to a computer. It enables the computer to produce sound for multimedia applications.

HDD (Hard Disk Drive)
An HDD is a non-volatile storage device that stores and retrieves digital information using rotating rigid platters coated with a magnetic material.

SSD (Solid State Drive)
An SSD is a storage device that uses flash memory to store data persistently. It provides faster access times and improved performance compared to traditional HDDs.

PSU (Power Supply Unit)
The PSU supplies electrical power to the computer. It converts the AC power from the wall outlet into DC power that the computer components can use.

Monitor
A monitor is an output device that displays visual information from the computer. It allows users to view images, videos, and other content.

Keyboard
A keyboard is an input device that allows users to enter text, commands, and other types of data into the computer.

Mouse
A mouse is an input device that allows users to move the cursor on the screen and interact with graphical user interfaces.

ROM (read-only memory)
ROM is a non-volatile memory type. This means it receives data and permanently writes it on a chip, and it lasts even after you turn off your computer.

Network Card
A network card, also known as a network interface card (NIC), enables a computer to connect to a network and communicate with other devices.

Optical Drive
An optical drive is a device that uses laser technology to read and write data from optical discs such as CDs, DVDs, and Blu-ray discs.

CPU Cooler
A CPU cooler is a device that helps dissipate heat generated by the CPU. It prevents the CPU from overheating and ensures stable performance.

Graphics Tablet
A graphics tablet, also known as a drawing tablet, is an input device that allows users to draw, sketch, and create digital artwork directly on the tablet surface using a stylus.

Speakers
Speakers are output devices that produce sound from the computer. They allow users to listen to audio, music, and multimedia content.

Webcam
A webcam is a video camera that captures video and audio data. It allows users to participate in video calls, conferences, and online streaming.

Printer
A printer is used to print documents, photos, and anything else that appears on your screen. There are many types of printers, including inkjet, laser, and photo printers. There are even all-in-one printers, which can also scan and copy documents.

Tips for Maintaining Computer Components

  • Regularly clean the computer components to remove dust and debris using compressed air or a soft cloth.
  • Ensure proper ventilation and airflow around the computer to prevent overheating.
  • Keep the computer in a clean and dust-free environment.
  • Update drivers and firmware for optimal performance and compatibility.
  • Install and regularly update antivirus software to protect against malware and viruses.
  • Back up important data regularly to prevent data loss.
  • Avoid placing drinks or liquids near the computer to prevent accidental spills.
  • Handle computer components with care, avoiding excessive force or static discharge.
  • Follow manufacturer's guidelines for maintenance and cleaning of specific components.

Troubleshooting Tips

  • If the computer doesn't power on, check if the power cable is properly connected and if the power supply switch is turned on.
  • If there is no display on the monitor, ensure that the monitor is connected to the computer and powered on. Check the cable connections and try a different cable if possible.
  • If the computer is running slowly, close unnecessary programs and processes. Run a malware scan and ensure that the computer has sufficient memory and storage space.
  • If there are no sounds from the speakers, check if the volume is muted or turned down. Verify that the speakers are properly connected to the audio output of the computer.
  • If a peripheral device is not working, check the cable connections, ensure that the drivers are up to date, and try connecting the device to a different port.
  • If the computer crashes or freezes, check for overheating by monitoring the temperature. Clean any dust from the components and ensure proper airflow. Update drivers and check for software conflicts.
  • If the internet connection is unstable, restart the modem and router. Check the network cables and verify that the network card drivers are up to date.
  • If a specific program or application is not working, try reinstalling or updating the software. Check for compatibility issues with the operating system or other components.
  • If there are strange noises coming from the computer, check for loose cables or components. It could indicate a problem with the fan, hard drive, or other hardware.
  • If you encounter frequent blue screen errors, note the error message and search for possible solutions online. It could be related to hardware or software issues.
